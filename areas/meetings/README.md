# Recurring Meetings

Ongoing meetings that you regularly attend and want to prep for and maintain context on. These are not 1:1s, not projects, and not one-off tasks — they're recurring forums with their own rhythm, attendees, and history.

## Structure

Each meeting gets a folder with:
- `README.md` — persistent context (attendees, cadence, shared doc link, standing agenda, what to bring)
- `sessions/` — dated prep and notes files (same pattern as 1:1s)
